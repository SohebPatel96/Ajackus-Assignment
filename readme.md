## Ajackus Assignment

First install all dependencies
```
composer install
```

Setup the database parameters in the .env file
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ajackus
DB_USERNAME=soheb
DB_PASSWORD=123456
```

After setting up the env file run
```
php artisan migrate
```



And then run ```php artisan serve```
